# Ko-Scheduler
A reading scheduler with support for the Kobo eReader (Kindles may be supported in the future). Automatically parses and sets up a customizable reading plan for any book and rewards you for maintaining a daily streak. Uses GTK+ (really terrible to work with :sob:) and is based off of [AmazinAxel (aka Alec, my broski)'s gtk+ build technique](https://github.com/AmazinAxel/ink-mail). 

Download, extract, and run the emulator binary to run the application on desktop devices, or extract and run the Kobo binary to run the application on Kobo eReaders.
